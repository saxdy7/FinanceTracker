### 7.5 **API Request/Response Flow**

**Authentication Flow:**
```
1. User Registration
   POST /api/v1/auth/register
   Body: {email, password, firstName, lastName}
   Response: {token, user}
   
2. JWT Token Storage
   Stored in localStorage
   Sent in Authorization header
   
3. API Request with Auth
   GET /api/v1/expenses
   Header: Authorization: Bearer <token>
   
4. Token Verification Middleware
   Extracts token
   Verifies signature
   Checks expiry (7 days)
   Adds user to request
   
5. Response
   Success: Data returned
   Error: 401 Unauthorized
```

**Error Handling:**
```javascript
Status Codes:
- 200: OK
- 201: Created
- 400: Bad Request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 500: Server Error

Error Response Format:
{
  error: true,
  status: 400,
  message: "Descriptive error message",
  code: "ERROR_CODE",
  timestamp: ISO8601
}
```
